# hey-clarissa

add ASCII art to your terminal. get your git stats. get a tip to get your commits up. check your horoscope. just a silly little pit stop for product designers who somehow ended up spinning dev environments from a terminal and want it to feel slightly more like theirs.

if that's your thing.

```bash
npm install -g hey-clarissa
clarissa
```

---

yes, there is another tool called clarissa out there — a much more serious, much better terminal agent. i found out after naming this one. i don't care. this one is for people who just want a horoscope and some ASCII art before they open their editor.
